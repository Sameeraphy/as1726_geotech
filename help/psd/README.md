# PSD Help

This folder contains support materials for the PSD tool.

## PSD Excel Template

- `psd_excel_template.xlsx` is a template you can use with the PSD GUI.
- The template includes two columns:
  - `Sieve (mm)`
  - `Retained (g)`

## How to use

1. Open `psd_excel_template.xlsx` in Excel.
2. Replace the sample rows with your own sieve and retained mass data.
3. Save the file.
4. In the PSD GUI, click **Load Excel** and select this file.
5. Click **Compute & Plot**.
